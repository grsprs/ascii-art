## [1.0.0] - 2024-01-XX

### Added
- ASCII art text rendering
- Support for standard, shadow, and thinkertoy banners
- Multi-line input support with `\n`
- CLI interface with flags
- File input/output support
- Input validation and size limits
- Cross-platform compatibility (Windows/Linux/macOS)
- Comprehensive test suite
- Golden file testing
- Security scanning and SBOM generation
- Performance benchmarks

### Security
- Input size limits (1MB default)
- Character validation (ASCII 32-126)
- Path traversal prevention
- Memory safety measures

## [0.2.0] - 2024-01-XX

### Added
- Shadow banner support
- Thinkertoy banner support
- File output functionality
- Enhanced error handling

### Changed
- Improved performance for large inputs
- Better error messages

### Fixed
- Memory leak in banner loading
- Edge case handling for empty input

## [0.1.0] - 2024-01-XX

### Added
- Basic ASCII art rendering
- Standard banner support
- CLI argument parsing
- Multi-line text support
- Basic test coverage
