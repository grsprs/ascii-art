# Banner Files

ASCII art font files for the generator.

## Available Fonts

- **standard.txt** - Classic ASCII art
- **shadow.txt** - Bold shadow effect
- **thinkertoy.txt** - Decorative style

## Format

- Height: 8 lines per character
- Characters: ASCII 32-126 (95 total)
- Order: Sequential (space to tilde)
- Separation: Single newline between characters
